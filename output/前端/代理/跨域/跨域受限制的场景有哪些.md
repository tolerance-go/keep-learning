# 跨域受限制的场景有哪些

- DOM 同源策略：禁止对不同源页面 DOM 进行操作。这里主要场景是 iframe 跨域的情况，不同域名的 iframe 是限制互相访问的

- ajax 同源策略：禁止使用 XHR 对象向不同源的服务器地址发起 HTTP 请求。

- 代理资源同源策略：禁止对不同源的代理缓存数据进行操作，比如 Cookie，LocalStorage 等无法读写
