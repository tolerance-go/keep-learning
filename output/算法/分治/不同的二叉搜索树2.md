# 不同的二叉搜索树2 [💻](https://github.com/tolerance-go/keep-learning/blob/master/src/%E7%AE%97%E6%B3%95%2F%E5%88%86%E6%B2%BB%2Findex.code.ts) [⛱️](https://github.com/tolerance-go/keep-learning/blob/master/src/%E7%AE%97%E6%B3%95%2F%E5%88%86%E6%B2%BB%2Findex.test.ts)

给定一个整数 n，生成所有由 1 ... n 为节点所组成的二叉搜索树。

示例:

输入: 3
输出:
[
  [1,null,3,2],
  [3,2,null,1],
  [3,1,null,null,2],
  [2,1,3],
  [1,null,2,null,3]
]
解释:
以上的输出对应以下 5 种不同结构的二叉搜索树：

   1         3     3      2      1
    \       /     /      / \      \
     3     2     1      1   3      2
    /     /       \                 \
   2     1         2                 3
