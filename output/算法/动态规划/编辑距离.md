# 编辑距离 [💻](https://github.com/tolerance-go/keep-learning/blob/master/src/%E7%AE%97%E6%B3%95%2F%E5%8A%A8%E6%80%81%E8%A7%84%E5%88%92%2F%E7%BC%96%E8%BE%91%E8%B7%9D%E7%A6%BB.code.ts) [⛱️](https://github.com/tolerance-go/keep-learning/blob/master/src/%E7%AE%97%E6%B3%95%2F%E5%8A%A8%E6%80%81%E8%A7%84%E5%88%92%2F%E7%BC%96%E8%BE%91%E8%B7%9D%E7%A6%BB.test.ts)

给你两个单词 word1 和  word2，请你计算出将 word1  转换成  word2 所使用的最少操作数

你可以对一个单词进行如下三种操作：

插入一个字符删除一个字符替换一个字符

示例  1：

输入：word1 = "horse", word2 = "ros" 输出：3 解释： horse -> rorse (将 'h' 替换为 'r') rorse -> rose (删除 'r') rose -> ros (删除 'e') 示例  2：

输入：word1 = "intention", word2 = "execution" 输出：5 解释： intention -> inention (删除 't') inention -> enention (将 'i' 替换为 'e') enention -> exention (将 'n' 替换为 'x') exention -> exection (将 'n' 替换为 'c') exection -> execution (插入 'u')
